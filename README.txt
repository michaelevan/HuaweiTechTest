1. Pengembangan Frontend
	- buka file index.html menggunakan browser anda
	- kosongkan salah satu inputan dan tekan tombol submit, maka terdapat pesan validasi sesuai kriteria

2. Pengembangan Backend
	- download & install node.js terlebih dahulu di browser
	- buka terminal dan ketik npm install express
	- ketik node server.js pada terminal
	- tekan ctrl + click pada url dari pesan yang terdapat di terminal

3. Automation Testing (perlu diingat harus menjalankan server.js terlebih dahulu)
	- buka terminal baru dan ketik npm install puppeteer
	- ketik node test.js pada terminal
	- lalu akan terbuka halaman web form pendaftaran dan diisi dengan inputan yang terdapat pada test.js
